Chemistry Simulations Suite - Special Projects 20

This is the page for my Special Projects 20 project this year (2025-2026), which was made over the course of about 8 months for school.

It is a collection of six simulations that each highlight a particular chemistry concept. This project is the first project I started and completed using raylib and C.

Click on the orange links below to get to the web builds of each project. The github repo is on the very bottom of the page.

NOTE: for SPECTROSCOPY, ignore the question marks: it is a visual bug from emscripten.
